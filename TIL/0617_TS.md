##### static
static 은 클라스에만 사용 가능
**static 이 붙은 것들은 인스턴스와는 무관하다**


예시: 
```
class Dog {
  static S = 1;
}
d = new Dog();
d.S        // Error! 
Dog().S  // 이것만 가능! 
```
근데 이제, Dog 클래스를 상속받은 다른 클래스에선 사용 가능: 
```
class Puppy extends Dog() {}
Puppy().S    // 이게 된다고? 확인 필요
```
