- 메소드는 어디 존재하는가?
  - prototype / 프로토타입
  - 다만, JS에서 모종의 이유로 obj 내에 있는 메소드 함수는 접근 가능하게 허용해줌
  - 예시:
```
class Dog {
  f1() {
    console.log('Dog.f1', this.name);
  }
}
// Dog.f1() <- 메소드 함수는 프로토타입으로 들어가기 때문에 바로 불러올 수 없다. (obj는 모종의 이유(sloppy mode)로 접근가능하게 해줌. strict 모드에선 안됨.)
Dog.prototype.f1();
// 출력값 ==> Dog.f1
```
